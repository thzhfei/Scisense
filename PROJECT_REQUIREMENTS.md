# 中科赛新智能技术有限公司网站项目需求文档

## 一、项目背景
中科赛新智能技术有限公司专注于人工智能、智能硬件和大数据领域的技术研发与产品应用。为提升公司在线形象、展示核心产品与技术优势、吸引客户及合作伙伴，需建设一个简洁、美观且功能完善的公司官方网站。

## 二、目标受众
- 潜在客户
- 合作伙伴
- 投资者
- 招聘人才

## 三、主要功能模块

### 1. 首页
- 公司简介（含愿景、使命）
北京中科赛新智能技术有限公司是一家专注于虚拟建模仿真、地理信息系统、通用人工智能应用等多个相关领域的高新科技公司。主要业务包括：虚拟建模仿真、地理信息系统开发、通用人工智能技术应用等。
公司与清华大学、北京航空航天大学、北京理工大学进行校企联合创新，并拥有一支由多学科、多领域的专业人才组成的研发团队，致力于以客户需求为导向，为客户提供高品质的数字化转型解决方案，确保您的需求得到满足。
- 核心产品/解决方案一览
- 新闻动态或最新公告
- 联系方式/二维码

### 2. 关于我们
- 企业文化
- 发展历程
- 团队介绍

### 3. 产品与技术
- 各类智能产品展示
- 技术特点与创新优势
- 案例应用

### 4. 解决方案
- 行业智能解决方案（例如：智能制造、智慧城市、医疗AI）
- 详细方案说明和优势

### 5. 新闻与资讯
- 公司新闻
- 行业动态
- 媒体报道

### 6. 招贤纳士
- 招聘职位列表
- 企业福利与工作环境
- 在线投递功能（简历邮箱或表单）

### 7. 联系我们
- 地址、电话、邮箱
- 在线留言表单
- 地图位置

## 四、技术需求
- 响应式网页设计，适配 PC 和移动端
- 前端使用 HTML5、CSS3、JavaScript，建议使用轻量框架（例如 Vue 或 React）
- 后端可选 Node.js/Express 或其他常用语言与框架，提供 API 支持留言和招聘投递
- SEO 优化，具备良好的访问速度
- 简洁、安全的内容管理方式，可通过 Markdown 或简单后台进行更新

## 五、视觉与风格
- 颜色方案参考公司LOGO，采用科技感蓝色调
- 扁平化、现代化设计风格
- 使用公司相关高清图片与图标

## 六、部署与维护
- 使用 HTTPS，证书配置
- 支持静态站点托管（如 GitHub Pages）或部署于云服务器（如阿里云/腾讯云）
- 定期备份内容
- 提供文档以便后续更新维护

## 七、项目交付
- 源代码仓库
- 详细的部署说明
- 用户使用手册
- 测试报告与验收说明
